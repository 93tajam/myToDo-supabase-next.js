### それぞれの技術の説明

- Next.js
    Reactを使ってアプリを作るためのフレームワーク。
    フレームワークとは言葉の通り枠組みで、Reactだけだとページ遷移とか自分で実装する必要があるがNext.jsならまとめて提供してくれる。

    https://qiita.com/tomy0610/items/f07d586c08a0a2aadb01
    
    どうやらやっぱり初心者用ではないがAI関連ツールが多いから使う必要ある。

- React
    UIをコンポーネント単位で組み立てるためのライブラリ。
    コンポーネントは部品なので、要は部品単位でボタンとか入力フォームを部品ごとに実装できる。だから再利用しやすい。

    ライブラリっていうのは、図書館のイメージで部品集。道具箱。

    UI部分特化。

- TypeScript
    JavaScriptに型を追加した言語。
    データ構造や関数の引数を明確にして、バグを減らす。

- Supabase
    PostgreSQLベースのBaaS、Backend as a Service。
    アプリケーションサーバー側がバックエンドのあれこれを肩代わりしてくれるという意味。
    認証、データベース、APIなどを素早く利用できる。

- Tailwind CSS
    ユーティリティファーストなCSSフレームワーク。クラス名を組み合わせて効率よくスタイリングできる。

Next.js（フロント）
    ↓
Supabase（サービス）
    ↓
PostgreSQL（DB）
    ↑
SQL（DB操作言語）
